# csci-2406-hw1
Homework 1 – ARM assembly without branch instructions

 CSCI-2406 Homework 1

 Description
ARM assembly program that calculates the sum of integers from m to n (inclusive) without using branch instructions.

 Registers
- r0 = m
- r1 = n
- r2 = sum (output)

 Technique
Uses conditional move instructions and manual PC modification instead of branch instructions.
